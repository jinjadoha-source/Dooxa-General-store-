
Wasiirka Wasaaradda Ganacsiga iyo Dalxiiska Jamhuuriyadda Somaliland Mudane:Maxamuud Xasan Sacad Saajin oo uu weheliyo Agaasimaha Guud ee Wasaaradda mudane: Xamse Xaamud Guuleed ayaa furay tababar
 Mar 08, 2022

Hargeysa.06/03/2022.Wasiirka Wasaaradda Ganacsiga iyo Dalxiiska Jamhuuriyadda Somaliland Mudane:Maxamuud Xasan Sacad Saajin oo uu weheliyo Agaasimaha Guud ee Wasaaradda mudane: Xamse Xaamud Guuleed ayaa furay tababar loo qabtay Ganacsatada JSL huteelka Damal Dahabshiil tababarkaas oo ku saabsanaa habka casriga ee lagu qaadan karo Liisamada Ganacsiga iyo habraacyada Sharciga ah ee loo marayo. Tababarkan waxa soo qaban qaabiyay Wasaaradda Ganacsiga iyo Dalxiiska oo ay ka dhex Muuqdaan Waaxda ITga ,Waaxda Ganacsiga iyo Waaxda Qorshaynta iyo Hay'adda SALSALOG.
